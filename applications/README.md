# Reference robot applications

This directory is reserved for independently buildable, complete GZRL reference
applications. Future applications may demonstrate chassis, gimbal, or other
robot configurations, including their device-tree overlays, Kconfig profiles,
control composition, and hardware-specific assembly.

Keep focused, single-capability examples in [`../samples/`](../samples/). Do not
add a partial robot application until it has a runnable end-to-end scope.
